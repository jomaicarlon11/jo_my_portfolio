Medical Records System

Project Information

  • Category: Web Development
  • Project Type: Personal
  • Project URL: https://jomai-jannel-2026-06-27.duckdns.org/_mr_jomai
  • Tech Stack Used: HTML, CSS, Bootstrap, JavaScript/jQuery, PHP Native, MySQL

Project Overview

The Medical Records Management System is a web-based healthcare platform designed to streamline the complete workflow of managing patient information, medical examination records, laboratory results, and company endorsements across multiple clinic branches. Built to support inter-branch operations, the system ensures that authorized users from different locations can securely access, update, and manage records through a centralized online platform.

The system features a secure login authentication module with configurable user types and access controls, ensuring that each user only sees and interacts with functions appropriate to their role. A visually intuitive dashboard with charts and analytics provides real-time insights into service utilization, helping clinics monitor the number of patients availing each service type across all branches.

One of the core components is the Company Endorsement Registration module, which automates the approval workflow and sends a soft copy of the approved endorsement directly to the company’s HR through email.

Allied health personnel staff can encode laboratory results and medical examination findings, then automatically generate a soft copy and printable of completed medical examination reports to reduced manual workloads. Also the system sends automated email notifications once their laboratory results or medical examination reports have been released to maintain efficient communication. The system includes also JO (Job Order) record filtering, as well as product list management for services or packages that patients can avail.

Secure Authentication & Access Control
